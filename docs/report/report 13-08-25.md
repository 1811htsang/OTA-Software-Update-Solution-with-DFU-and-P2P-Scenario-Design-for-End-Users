Dựa trên các báo cáo ngày 30-07-25

Kiến trúc hệ thống vẫn được xác định giữ nguyên và đang hoàn thiện đúng theo xác định

Đã hoàn tất chuyển đổi sử dụng từ Arduino IDE sang ESP-IDF

Về phần cấu hình thư mục cho SD card trên ESP32, đề xuất như sau
- 1 file `ls.txt` để thực hiện list các file trên FTP server
- 1 file `base_firmware.bin` để lưu lại firmware gốc cho kịch bản giả định firmware mới cập nhật sẽ không thành công
- 1 folder `firmware_updates` để chứa các bản firmware mới cập nhật

Về phần cấu hình MQTT message format, nên bắt đầu thiết kế triển khai sau khi hoàn tất các bước trên.

Về vấn đề bảo mật hiện tại không nên vội triển khai mà đảm bảo các chức năng cơ bản hoạt động ổn định trước, sau đó mới triển khai bảo mật.
