# Ghi chú bổ sung

# Thiết kế topic cho MQTT
`project_name/device_id/direction` 

Trong đó:
- `project_name`: Tên dự án (ví dụ: `NCKHSV`)
- `device_id`: Địa chỉ MAC của thiết bị (ví dụ: `ESP32_MAC_ADDRESS`)
- `direction`: Hướng của gói tin (ví dụ: `request` hoặc `response`)

## Cấu trúc hóa gói tin

### Gói tin yêu cầu
```json
{
  "deviceId": "ESP32_MAC_ADDRESS",
  "chipId": "0x0000FAF4",
  "requestType": "UPDATE_CHECK",
  "prevUpdatedDate": "2023-12-25"
} // may split 
```

### Yêu cầu không hợp lệ từ gateway
```json
{
  "status": "BAD_REQUEST",
  "message": "Invalid request format"
}
```

### Yêu cầu không hợp lệ requestType từ gateway
```json
{
  "status": "BAD_REQUEST",
  "message": "Invalid requestType"
}
```

### Gói tin xác thực thiết bị
```json
{
  "deviceId": "ESP32_MAC_ADDRESS",
  "chipId": "0x0000FAF4",
  "requestType": "CHECK_INFO"
}
```

### Phản hồi khi server gặp lỗi
```json
{
  "status": "SERVER_UNEXPECTED_ERROR",
  "message": "Server cant respond to validation request, try again later"
}
```

### Phản hồi tìm thấy thiết bị
```json
{
  "validationResult": "VALID",
  "suitableFirmware": [
    "firmware_v1.bin",
    "firmware_v2.bin"
  ],
  "currentAvailableFirmware": [
    "firmware_v1.bin",
    "firmware_v2.bin"
  ] 
}
```

suitableFirmware: Danh sách firmware phù hợp với thiết bị dựa trên chipId và các tiêu chí khác.

allAvailableFirmware: Danh sách tất cả firmware có sẵn trên server, dùng để tham khảo.

### Phản hồi không tìm thấy thiết bị
```json
{
  "validationResult": "INVALID",
  "suitableFirmware": null
}
```

### Phản hồi không hợp lệ từ server
```json
{
  "status": "DEV_VAL_FAILED",
  "message": "Server failed to validate device, refuse to proceed with update request"
}
```

### Phản hồi hợp lệ từ server
```json
{
  "status": "DEV_VAL_SUCCESS",
  "message": "Server validated device successfully"
}
```

### Phản hồi danh sách firmware phù hợp
```json
{
  "status": "FW_LIST_SUCCESS",
  "firmwareList": [
    "firmware_v1.bin",
    "firmware_v2.bin"
  ]
}
```

### Gói tin gửi lựa chọn cập nhật
```json
{
  "deviceId": "ESP32_MAC_ADDRESS",
  "chipId": "0x0000FAF4",
  "isValidate": true,
  "requestType": "UPDATE_CONFIRM",
  "requestFile": "new_firmware_v2.bin"
}
```

### Phản hồi không có cập nhật
```json
{
  "status": "NO_UPDATE",
  "message": "Device is running the latest version.",
}
```

### Phản hồi có cập nhật
```json
{
  "status": "UPDATE_AVAILABLE",
  "fileName": "new_firmware_v2.bin",
  "downloadInfo": {
    "server": "192.168.0.101",
    "path": "/FTP_Site/"
  }
}
```


### Phản hồi tìm firmware trên Gateway
```json
{
  "status": "FW_FOUND",
  "message": "Found firmware inside gateway storage"
}
```

### Phản hồi không tìm thấy firmware trên Gateway
```json
{
  "status": "FW_NOT_FOUND",
  "message": "No firmware found inside gateway storage, continue to check inside server storage"
}
```
